NotificationManager()Constructor
public function NotificationManager(defaultStyle:String, defaultNotificationImage:String, defaultCompactNotificationImage:String, notificationSound:String = null, displayLength:int, displayLocation:String)

Constructor to create a valid NotificationManager object. Will create a notification engine with the passed in values as defaults settings. Can also change these settings afterwards, but the first three at least (defaultStyle, defaultNotificationImage, and defaultCompactNotificationImage) are required. The rest are optional and can be omitted or left null.

Parameters
defaultStyle:String — Location of compiled stylesheet to use as default style.
 
defaultNotificationImage:String — Location of 50x50 image to use as default notification image when one isn't specified.
 
defaultCompactNotificationImage:String — Location of 16x16 image to use as default compact notification image when one isn't specified.
 
notificationSound:String (default = null) — (Optional) Location of notification sound to use when displaying a notification. Leave null if no sound desired. Defaults to null.
 
displayLength:int (default = NaN) — (Optional) Length, in seconds, that notifications will stay on the screen. Defaults to NotificationConst.DISPLAY_LENGTH_MEDIUM.
 
displayLocation:String (default = NaN) — (Optional) Location on-screen (i.e. top-left, top-right, bottom-left, or bottom-right) where notifications are to appear. Defaults to NotificationConst.DISPLAY_LOCATION_AUTO.

show()method 
public function show(notificationTitle:String, notificationMessage:String, notificationImage:String, notificationLink:String = null, isCompact:Boolean = false, isSticky:Boolean = false, isReplayable:Boolean = true):void

Method to show notification using raw, passed-in values. This is really a convenience method for quickly showing a notification. Behind the scenes, it simply creates a notification object and calls the showNotification() API with that object.

Parameters

notificationTitle:String — The title for the notification pop-up.
 
notificationMessage:String — The message for the notification pop-up.
 
notificationImage:String — (Optional) The URL, if any, to direct the user to when the notification is clicked. If null, there will be no action on notification click. Defaults to null.
 
notificationLink:String (default = null) — (Optional) Parameter to set whether or not the notification should be displayed as a compact notification. If this is true, the notification message will not be displayed. Defaults to false.
 
isCompact:Boolean (default = false) — (Optional) Parameter to set whether or not the notification is sticky and should remain on-screen until the user manually closes it. Defaults to false.
 
isSticky:Boolean (default = false) — (Optional) Parameter to set whether or not the notification is replayable, meaning whether or not it will show up when the replayLatestFiveNotifications() API is invoked. This is most often set to true, but would be set to false for certain types of notifications such as system notifications. Defaults to true.
 
isReplayable:Boolean (default = true)
